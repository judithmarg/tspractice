# Instrucciones para ejecutar el proyecto

- Clonar repositorio
  
- Usar comandos:

```console
 npm install
 npm run dev
```

## Lista de funcionalidades implementadas

- Visualizar todos los usuarios registrados
 
- Agregar nuevos usuarios al sistema

- Buscar usuarios por nombre (es requerido todo el nombre)
 
PERSISTENCIA DE DATOS

- Los datos deben mantenerse al recargar la página

- Implementar almacenamiento local en el navegador

INTERFAZ DE USUARIO

- Vista principal: Lista de usuarios con capacidad de búsqueda

- Formulario de registro: Para agregar nuevos usuarios

- Navegación entre las diferentes vistas
